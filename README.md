# Сегментация клиентов онлайн-магазина подарков

Проект по анализу и сегментации клиентов на основе RFM-анализа с использованием различных алгоритмов кластеризации.

## Описание проекта

**Бизнес-задача:**  
Произвести сегментацию существующих клиентов онлайн-магазина подарков, проинтерпретировать полученные сегменты и предложить стратегию взаимодействия с каждым из них.

**Техническая задача:**  
Построить модель кластеризации клиентов на основе их покупательской способности (Monetary), частоты заказов (Frequency) и срока давности последней покупки (Recency).

### Основные этапы проекта:
1. Предобработка данных (очистка, обработка возвратов, специальные транзакции)
2. Разведывательный анализ данных (EDA)
3. Создание RFM-таблицы
4. Генерация эмбеддингов описаний товаров (SentenceTransformers)
5. Кластеризация с использованием нескольких алгоритмов:
   - KMeans
   - Agglomerative Clustering
   - Gaussian Mixture Models
6. Сравнение моделей и выбор лучшей
7. Интерпретация кластеров и разработка маркетинговых стратегий
